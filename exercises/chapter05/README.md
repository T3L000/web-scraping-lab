# Chapter 05 Exercises

目标：练习更复杂的 HTML 解析，包括标签选择、属性读取、树结构导航和正则辅助提取。

## Notebooks

- `bs_find_all_filters.ipynb`: `find_all`、`class` / `string` 等筛选参数练习。
- `bs_tree_navigation.ipynb`: `children`、`descendants`、`siblings`、`parent` 等树结构导航练习。
- `bs_regex_images.ipynb`: 用正则筛选图片 `src` 属性。
- `regex_scratch.ipynb`: 正则临时实验草稿。
